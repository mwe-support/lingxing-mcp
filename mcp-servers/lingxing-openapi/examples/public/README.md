# Public Examples

This folder contains sanitized public examples only.

- `gateway.env.example`: example gateway environment file
- `tokens.example.json`: example multi-member token file
- `claude_code.mcp.json`: example project-level MCP config for stdio mode
- `cursor.mcp.json`: example project-level MCP config for Cursor
- `claude_desktop_http_config.json`: example HTTP MCP config for user-level clients

All values are placeholders.
