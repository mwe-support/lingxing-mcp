# 拉群引导草案

## 短版

如果你正在用领星 ERP，且希望把领星数据接进 MCP、Claude Code、Codex、Cursor 这类 AI 工作流，欢迎进群交流。

群里适合讨论的方向：

- 固定出口网关部署
- 领星白名单问题
- 团队共享接入
- MCP 客户端配置
- 只读数据分析工作流
- 产品表现、广告报表、利润和源表怎么接
- Amazon 报告导出和下载解析
- ASIN 周报 / 广告周报 / 经营周报补数

入群前最好准备：

- 你现在的使用场景
- 是否有云服务器
- 是否已经申请领星 OpenAPI
- 你最关心的是销量/订单，还是广告/流量/利润/库存/报告导出
- 当前卡在哪一步

## 长版

这个群不主打泛泛聊天，主要适合已经有明确需求的人：

- 想把领星 OpenAPI 接进 AI 工具
- 想解决团队多人共享访问
- 想做固定出口网关
- 想在不同 IDE 中稳定调用领星数据
- 想把广告、流量、利润、库存、补货、报告导出也统一纳入同一个只读入口
- 想用领星数据去补飞书周表、经营周报或内部分析报表

群内更欢迎这三类人：

- 正在用领星的运营团队
- 做亚马逊数据流和 AI 工作流的人
- 对 MCP 接入和团队部署感兴趣的人

如果只是想要一个“现成代运营方案”，这可能不是最适合的群。  
如果你是想自己把这套东西真正搭起来，那会非常适合。
